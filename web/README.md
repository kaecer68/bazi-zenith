# 八字命盤管理應用前端

基於 React + TypeScript + Tailwind CSS 的八字排盤管理界面

## 功能特性

### 1. 多人生辰管理
- 新增、編輯、刪除多個人的生辰資料
- 本地儲存（LocalStorage）持久化
- 搜尋功能快速查找資料
- 點選即可載入進行排盤

### 2. 曆法支援
- **陽曆（西曆）**：標準公曆日期輸入
- **農曆（陰曆）**：傳統農曆日期輸入，自動轉換為陽曆進行計算

### 3. 完整八字排盤輸出

#### 四柱資訊
| 項目 | 內容 |
|------|------|
| 天干 | 甲、乙、丙、丁...等十干 |
| 地支 | 子、丑、寅、卯...等十二支 |
| 十神 | 比肩、劫財、食神、傷官、正財、偏財、正官、七殺、正印、偏印 |
| 藏干 | 地支藏本氣、中氣、餘氣及其十神 |
| 納音 | 六十甲子納音五行（如：海中金、爐中火） |
| 長生十二運 | 長生、沐浴、冠帶、臨官、帝旺...等十二階段 |
| 神煞 | 天乙貴人、祿神、羊刃、文昌、驛馬、桃花、華蓋、將星、紅鸞、天喜 |

#### 命理分析
- **身強身弱評估**：得令、得地、得助三得分析
- **喜忌五行**：喜用神與忌神標示
- **吉方位**：財位、事業位、文昌位、桃花位
- **起運歲數**：計算開始進入大運的年齡

#### 大運與流年
- **大運排列**：每十年一大運的完整列表
- **流年斷語**：針對目標年份的運勢分析
  - 值太歲/沖太歲檢測
  - 神煞入流年（貴人、桃花、驛馬）
  - 身強身弱與流年關係
  - 運勢吉凶評估

## 啟動方式

### 前置需求
1. 後端服務必須先啟動（在專案根目錄執行）
```bash
cd /Users/kaecer/workspace/bazi-zenith
make run-rest
```

### 前端開發模式
```bash
cd web
npm install    # 首次安裝依賴
npm run dev    # 啟動開發伺服器
```

應用將在 http://localhost:5173 啟動

### 生產建構
```bash
cd web
npm run build  # 建構到 dist/ 目錄
```

## API 整合

前端透過代理設定連接後端 REST API：
- 開發時：`http://localhost:8080`
- 生產環境：配置對應的 API 端點

### 主要 API 端點
```
POST /api/v1/chart    # 生成八字命盤
GET  /health          # 健康檢查
```

## 使用流程

1. **新增生辰資料**
   - 點擊「新增」按鈕
   - 輸入姓名、選擇性別
   - 選擇曆法類型（陽曆/農曆）
   - 輸入年月日時
   - 點擊「排盤計算」

2. **管理現有資料**
   - 左側列表顯示所有儲存的生辰
   - 點擊載入，點擊垃圾桶圖示刪除
   - 搜尋框可按姓名篩選

3. **查看排盤結果**
   - 四柱八字表格完整展示
   - 身強身弱量化分析
   - 喜忌五行與吉方位
   - 大運排列與流年斷語

## 設計理念

### 科學性
- 所有數據基於天文計算（節氣、太陽黃經）
- 身強身弱採用量化評分模型
- 清晰標示計算邏輯與依據

### 友善介面
- 響應式設計支援桌面與移動設備
- 傳統中式配色（紅、金、墨）
- 資訊分層展示，避免過度擁擠
- 圖示輔助理解

### 完整性
- 展示所有後端支援的數據欄位
- 幫助開發者理解 API 支援範圍
- 便於發現需要擴展的功能邊界
